Javascript

Javascript: Is a language that has nothing to do with Java despite it's name, official name is ECMAScript. Javascript is a programming language used to make webpages interactive.

Operators

Operators are the actions that can be performed with variables using mathematical operations as well as text concatention.

Variables

A variable is something that can hold a bit of data that has been manipulated by the program. It should be written in 'lower camel case', is declared by the term 'var'.

Functions

Functions: A function performs an action, can have an arguments passed into it, and is not provided by the language but by the programmer.

Methods

Methods: Actions that can be preformed on objects. They are something that is provided by the language, such as .toUppercase, .charAt, etc.

Front End Logic

Front End Logic or User Interface: This is what the user interacts with in the browser, as well as the visual display. This includes, and is not limited to drop-down menus, buttons, tabs, etc.

Backend Logic

Backend Logic or Business Logic: This is the part of the browser that is "static" in other words, we do not interact with backend logic. Examples of this would include Databases or mathematic functions.

DOM

DOM or Document Object Model: Can be seen in the developer tools in the browser, it is your browsers interpretation of the HTML it reads.

jQuery

jQuery: A JavaScript library (lots of prewritten JavaScript) that makes it simpler to make web pages interactive.

Strings

Strings: Values made up of text, surrounded by quotes, can be made up of letters, numbers, symbols, and punctuation. (*To put a quote inside a string use single quotes surrounding the quote, as well as quotes around the string.)
